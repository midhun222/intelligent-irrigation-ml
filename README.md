# intelligent-irrigation-ml

# Intelligent Irrigation System using Machine Learning

# About the Project
This project is about building a simple machine learning model that decides whether an irrigation pump should be turned ON or OFF based on soil moisture and temperature.

The idea is to make irrigation smarter so that water is used only when needed.

---

# What I Did
- Loaded and explored the dataset
- Checked for missing values and basic statistics
- Converted the crop column into numerical format
- Applied log transformation to improve data distribution
- Visualized the data using graphs
- Split the data into training and testing sets
- Trained a Logistic Regression model
- Evaluated the model using accuracy and other metrics

---

# Model Used
I used Logistic Regression because it works well for binary classification problems like this (pump ON/OFF).

---

# Result
The model achieved very high accuracy (around 100%), which shows it can predict irrigation decisions effectively.
